# Xlog

E[X]tend [Log]rus is Wrapper function for use logrus as builder design pattern and intergrate to elasticsearch in one line of code.

## Geting Started

> go get github.com/pepodev/xlog

```go
package main

import (
    "github.com/pepodev/xlog"
)

func main(){
    xlog.Debugln("Debug")
}
```
